# Documentación de Componentes

Esta carpeta contiene la documentación detallada de todos los componentes y directivas de Catarina, organizados por categorías en directorios separados.

## Estructura de Directorios

```
components/
├── design/          # Componentes de diseño básicos
├── forms/           # Componentes de formulario
├── panels/          # Componentes de paneles
├── overlays/        # Componentes de superposición
└── directives/      # Directivas
```
